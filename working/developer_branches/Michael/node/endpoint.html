WIldFire API

Wildfire API Endpoints

  • /detail - Returns wildfire data at lowest granulatiry. Fire incident level.
  • /county_codes - Retunrs list of California County Codes. Overall code included for programming purposes.
  • /yearly_stats_detail - Returns wildfire metrics summarized by year
  • /yearly_stats_county - Returns wildfire metrics summarized by year by county, including overall total.
  • /yearly_stats_incident_level - Returns wildfire metrics summarized by year, county and incident level. Includes Overall data
  • /overall_stats - Returns summary wildfire stats for all years. One record..
